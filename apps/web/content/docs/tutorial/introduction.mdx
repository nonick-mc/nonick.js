---
title: Botを導入する
---

import { Step, Steps } from 'fumadocs-ui/components/steps';

<Callout type="info">
Botをサーバーに招待するには、`サーバーの管理`権限を所有している必要があります。
</Callout>

<Steps>
<Step>

### ダッシュボードにアクセスする
[ダッシュボード](https://dashboard.nonick-js.com)にアクセスし、Discordアカウントを使用してログインを行います。

</Step>
<Step>

### 「サーバーを追加」ボタンをクリックする
![ダッシュボード](https://cdn.nonick-js.com/docs/tutorial/dashboard.png)

</Step>
<Step>

### 要求された権限を確認し、認証する
招待リンクにアクセスし、どのサーバーに追加するか選択すると、Botが必要としている権限の一覧が表示されます。
「認証」を選択すると、Botがサーバーに参加し、NoNICK.jsを使用できるようになります。

<Callout type="error">
  要求された権限は、**全てBotの動作に必要な権限です**！権限を剥奪すると、一部機能の使用が制限される場合があります。
  また、更新の度に要求する権限が増える場合があります。その際はこの手順をもう一度繰り返すことで、Botを退室させることなく権限を更新することができます。
</Callout>
![権限の確認画面](https://cdn.nonick-js.com/docs/tutorial/permission_modal.png)

</Step>
</Steps>
